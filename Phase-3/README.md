# Phase 3: Model Evaluation and Testing

## Objective
The objective of this phase is to select the best model based on the evaluations and test it on unseen data.

## Models Used
- K-Nearest Neighbors (KNN)
- Logistic Regression
- Decision Tree
- Random Forest

## Evaluation Methods
The models were evaluated using:
- Accuracy Score
- Confusion Matrix
- Classification Report (Precision, Recall, F1-score)

## Model Comparison
All models were compared based on their performance metrics. Random Forest performed the best among all models.

## Final Model Selection
Random Forest was selected as the final model because it provided:
- Highest accuracy
- Balanced precision and recall
- Fewer misclassifications

## Model Testing
The final model was tested using:
- Samples from the test dataset
- Both normal and attack cases

The model successfully predicted unseen data and demonstrated good generalization.

## Conclusion
The project successfully identified the best model for network intrusion detection. Random Forest proved to be the most reliable model based on evaluation and testing.
